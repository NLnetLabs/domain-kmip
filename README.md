# `domain-kmip`: KMIP HSM signing for [`domain`]

[`domain`]: https://github.com/NLnetLabs/domain

`domain-kmip` integrates KMIP (the Key Management Interoperability Protocol)
with [`domain`]'s support for signing DNS records/zones.

**Warning:** This library is currently in `v0.0.x` versioning, where breaking
changes will be made often, including on the `main` branch.  Please wait to use
it until it reaches `v0.1.0` or later.
